Simple-Layout basic tutorial.

Simple Layout is an easy way to layout app/game assets without caring about what is the final resolution. In this tutorial I will show you how to create a welcome screen with a menu for a memory game. You can view the result here, resize the browser window to see how Simple-Layout deals with any given size. You can get the full source code here.

In this tutorial I will be using PixiJs, so we'll start by creating a simple index.html:
